# Design Update: Minimalist Black/Grey/White Theme

## Summary of Changes

The website has been updated to a minimalist aesthetic with a monochrome color palette and clean sans serif typography.

## Color Palette

### Previous (Colorful)
- Primary: #2c3e50 (Blue-grey)
- Secondary: #e67e22 (Orange)
- Accent: #3498db (Blue)

### New (Minimalist)
- Primary: #000000 (Black)
- Secondary: #333333 (Dark Grey)
- Accent: #666666 (Medium Grey)
- Light Grey: #f5f5f5
- White: #ffffff

## Typography

### Previous
- Headings: Playfair Display (Serif)
- Body: Inter (Sans Serif)

### New
- All Text: System Sans Serif Stack
  - macOS: San Francisco
  - Windows: Segoe UI
  - Others: Helvetica Neue, Arial

## Design Elements Updated

### Removed
✓ Rounded corners (border-radius: 0 everywhere)
✓ Heavy box shadows (minimal shadows now)
✓ Color accents (orange, blue)
✓ Decorative fonts
✓ Google Fonts dependency

### Minimized
✓ Hover animations (subtle scale instead of dramatic)
✓ Border thickness (2px → 1px)
✓ Button padding (more compact)
✓ Shadow intensity (lighter, softer)

### Style Changes

**Navigation**
- Thinner underline (1px vs 2px)
- Black text instead of blue-grey
- Cleaner, simpler logo

**Hero Section**
- Black gradient overlay instead of colored
- Minimalist button with border outline
- Reduced letter spacing
- System fonts

**Buttons & Filters**
- Square corners (no border-radius)
- Black on white color scheme
- Thinner borders
- Less padding

**Cards (Writings, Recommendations, Gallery)**
- No rounded corners
- Minimal shadows
- Thin borders instead of heavy shadows
- Subtle hover effects (2% scale vs 10%)

**Tags & Badges**
- Transparent background with borders
- Black text on white
- Square corners
- Smaller, more compact

**Footer**
- Black background
- White text with minimal accents
- Outlined newsletter button

## Visual Result

The site now has a:
- Clean, editorial look
- Modern minimalist aesthetic
- Professional, timeless design
- Better readability
- Faster loading (no external fonts)
- Platform-native feel

## Browser Performance

**Improvements:**
- No Google Fonts loading
- Faster initial render
- Better font rendering (native system fonts)
- Reduced CSS complexity
